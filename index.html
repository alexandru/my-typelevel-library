Contributing

The My Typelevel Library project welcomes contributions from anybody wishing to participate. All code or documentation that is provided must be licensed with the same license that My Typelevel Library is licensed with (Apache 2.0, see LICENCE).

People are expected to follow the Scala Code of Conduct when discussing My Typelevel Library on GitHub, Gitter channel, or other venues.

Feel free to open an issue if you notice a bug, have an idea for a feature, or have a question about the code. Pull requests are also gladly accepted. For more information, check out the contributor guide.
